Trivia Game (projet 1 Ironhack)

Contexte

Ce projet a été réalisé lors du bootcamp Ironhack, en 4 ou 5 jours.

Il nous était demandé de développer un jeu, en front-end, pour mettre en application ce que nous venions d'apprendre.


Procédé

Pour cet exercice j'ai utilisé du HTML, du CSS, du JavaScript avec de la manipulation DOM, ainsi qu'Axios et une API (OpenTriviaDB).

J'ai créé une page d'accueil qui permet de choisir le thème et la difficulté du quiz. Lorsqu'on choisit, on a 4 réponses possibles et un certain temps pour répondre à la question. Si l'on ne répond pas dans les temps, le quiz passe à la question suivante. Il y a un système de comptage de points et le résultat final varie en fonction du nombre de points obtenus.

Vous pouvez cliquer sur l'image pour voir la page ou visualiser le code dans ce repository (dossier GitHub).
