Page d'accueil de Google

Contexte

Cet exercice a été réalisé dans le cadre des cours de The Odin Project (curriculum "Web Development 101").

Il fallait reproduire la page d'accueil de Google : avoir le même agencement et le même contenu, peu importe si les liens et la recherche ne fonctionnent pas.

Nous devions donc construire une page qui contienne :

  • le logo Google
  • la barre de recherche et les boutons
  • le header (fixe)
  • le footer

Vous pouvez consulter les instructions complètes directement sur The Odin Project .


Procédé

Pour cet exercice j'ai utilisé du HTML et du CSS. J'ai utilisé une CSS Grid et plusieurs flexboxes pour rendre le tout responsive. J'ai essayé de faire en sorte que ma page soit la plus ressemblante possible.

Vous pouvez visualiser le code dans ce repository.
